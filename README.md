# nfl-lakehouse-azure
A production-grade data lakehouse built on Azure Databricks, using NFL play-by-play data from nflfastR. Covers the full stack: infrastructure as code with Terraform, medallion architecture on Delta Lake, PySpark transformation pipelines, MLflow-tracked ML models (game outcome prediction, win probability, player clustering), and a Databricks SQL analytics dashboard. Architectural decisions documented as if this were an enterprise FSO deployment.
